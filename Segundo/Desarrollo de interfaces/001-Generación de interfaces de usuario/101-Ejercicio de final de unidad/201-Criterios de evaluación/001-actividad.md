Ejercicio de final de unidad 1:

El objetivo consiste en poner en práctica los conocimientos adquiridos en la unidad 1 de desarrollo de interfaces. Crea un proyecto mediante el cual trabajes uno o varios componentes de interfaces de usuario (en principio, interfaces empresariales).

Selecciona controles de formulario, comprueba cuales no existen actualmente, y desarrolla una librería de componentes de interfaz de usuario personalizados y reutilizables.
