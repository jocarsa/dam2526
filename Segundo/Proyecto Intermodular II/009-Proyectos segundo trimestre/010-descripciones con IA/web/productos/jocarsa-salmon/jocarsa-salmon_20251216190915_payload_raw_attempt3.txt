   - Las salas se almacenan en una base de datos SQL.

3. **Visualización de Salas:**
   - Muestra las salas existentes en una tabla con opciones para editar o eliminar cada sala.

4. **Formulario de Inicio de Sesión:**
   - Incluye un formulario para iniciar sesión y muestra mensajes de error si el ingreso es incorrecto.
