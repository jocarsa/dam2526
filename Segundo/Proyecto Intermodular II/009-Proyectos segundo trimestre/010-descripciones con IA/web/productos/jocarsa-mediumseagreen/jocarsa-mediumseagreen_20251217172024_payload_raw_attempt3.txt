### Lenguajes de Marcas

**HTML (Hypertext Markup Language):**
- **Descripción:** Es el lenguaje de marcas más conocido y ampliamente utilizado para crear páginas web. Permite a los desarrolladores agregar elementos como imágenes, enlaces y texto a una página.
- **Uso en Contexto:**
  ```html
  <h1>Encabezado de nivel uno</h1>
  <p>Texto normal del párrafo.</p>
  ```

**XML (Extensible Markup Language):**
- **Descripción:** Es un lenguaje de marcas utilizado para crear documentos estructurados. Se utiliza comúnmente en la creación de formatos de intercambio de datos y en la descripción de esquemas de negocio.
- **Uso en Contexto:**
  ```xml
  <book>
    <title>El Lobo de Wall Street</title>
    <author>William Martínez</author>
    <year>2023</year>
  </book>
  ```

**Markdown:**
- **Descripción:** Es un lenguaje de marcas ligero y fácil de leer, utilizado para crear contenido en línea. Permite a los usuarios agregar elementos como encabezados, listas y enlaces a su contenido.
- **Uso en Contexto:**
  ```markdown
  # Encabezado de nivel uno

  ## Encabezado de nivel dos

  - Lista no ordenada
    - Item 1
    - Item 2

  [Enlace](http://ejemplo.com)
  ```

### Explicación

- **HTML:** Es el lenguaje de marcas más conocido y ampliamente utilizado para crear páginas web. Permite a los desarrolladores agregar elementos como imágenes, enlaces y texto a una página.
- **XML:** Es un lenguaje de marcas utilizado para crear documentos estructurados. Se utiliza comúnmente en la creación de formatos de intercambio de datos (XSD) y en la descripción de esquemas de negocio.
- **Markdown:** Es un lenguaje de marcas ligero y fácil de leer, utilizado para crear contenido en línea. Permite a los usuarios agregar elementos como encabezados, listas y enlaces a su contenido.

### Actividades Prácticas (Paso a Paso, Con Soluciones o Pistas Breves)

1. **HTML:**
   - **Tarea:** Crea una página web simple con un encabezado y parágrafo.
   - **Solución:** 
     ```html
     <!DOCTYPE html>
     <html lang="es">
     <head>
       <meta charset="UTF-8">
       <title>Página Web Simple</title>
     </head>
     <body>
       <h1>Encabezado de nivel uno</h1>
       <p>Texto normal del párrafo.</p>
     </body>
     </html>
     ```

2. **XML:**
   - **Tarea:** Crea un documento XML con información básica sobre un libro.
   - **Solución:** 
     ```xml
     <?xml version="1.0" encoding="UTF-8"?>
     <book>
       <title>El Lobo de Wall Street</title>
       <author>William Martínez</author>
       <year>2023</year>
     </book>
     ```

3. **Markdown:**
   - **Tarea:** Escribe un breve resumen en formato Markdown.
   - **Solución:** 
     ```markdown
     # Resumen

     Este es un ejemplo de cómo escribir un resumen en Markdown.

     El resumen abarca varios aspectos importantes y es estructurado de manera clara.
     ```
