¿Por qué Logos?

Indentación y legibilidad

Bloques definidos por espacios. Sin llaves ni punto y coma.

Léxico greco-latino

Palabras reservadas en minúsculas, sin acentos ni caracteres especiales.

Núcleo estable

El núcleo es pequeño y explícito. El resto vive en librerías/extensiones.

Minimalista y expresivo

Asignación directa, control de flujo claro y una stdlib conceptual.

Ejemplos de sintaxis

Hola mundo

import sonus
sonus.dic("salve mundus!")

Condicionales

import sonus

nomen = "Logos"

si nomen == "Logos":
    sonus.dic("optime!")
alio:
    sonus.dic("quis es?")

Operadores lógicos

import sonus

x = 7

si x > 0 et x < 10:
    sonus.dic("en rango")
alio:
    sonus.dic("fuera de rango")

Bucles: pro + series

import sonus

pro i in series(0, 5):
    sonus.dic(i)

Bucles: dum + perge/frange

import sonus

x = 0
dum x < 10:
    x = x + 1
    si x == 2:
        perge
    si x == 5:
        frange
    sonus.dic(x)

Funciones: munus + redit

import sonus

munus quadratum(x):
    redit x * x

sonus.dic(quadratum(9))

Más ejemplos

Literales: verum/falsum/nulla

import sonus

activo = verum
valor = nulla

si activo et valor == nulla:
    sonus.dic("estado valido")
alio:
    sonus.dic("estado mixtus")

Entrada/salida: sonus.lege

import sonus

nomen = sonus.lege("nomen? ")
sonus.dic("salve " + nomen)

Excepciones: conare/nisi/denique

import sonus

conare:
    x = divide(10, 0)
nisi:
    sonus.dic("error")
denique:
    sonus.dic("fin")

Lanzar excepción: iacta

import sonus

munus valida(x):
    si x == nulla:
        iacta Error("valor invalidus")
    redit x

FAQ

¿Por qué palabras greco-latinas?

Para lograr un léxico distintivo, consistente y sin dependencia total del inglés, manteniendo legibilidad.

¿Por qué import está en inglés?

Se conserva por ser un estándar internacional consolidado y familiar para la mayoría de programadores.

¿Hay declaración de variables?

No. Una variable existe al asignarle un valor: x = 10.

¿Dónde está la librería estándar?

El núcleo define el lenguaje. La stdlib se modela como módulos (por ejemplo sonus y series).
