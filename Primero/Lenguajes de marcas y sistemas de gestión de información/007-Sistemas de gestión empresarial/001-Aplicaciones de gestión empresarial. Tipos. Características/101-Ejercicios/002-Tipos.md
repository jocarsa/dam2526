Yo no creo mucho en los tipos de software empresarial
No creo en las etiquetas, porque en la práctica, una empresa tiene una necesidad, y tu le programas lo que sea que sea necesario para resolver esa necesidad.

No está mal clasificar los software de gestión (sin tomarnos demasiado en serio esa clasificación, es decir, es orientativa)

Muy horizontales - transversales a cualquier empresa

CRM - Customer RelationShip Management - Sistema de gestión de la relación con el cliente (ventas)

ERP - Enterprise Resource Planning (Sistema de planificación integral de la empresa) - Software que cubre todos los aspectos de una empresa - es el mayor software que existe - suele ser un conjunto de módulos orquestados para trabajar en conjunto

Facturación/contabilidad - Encargan de facturar, llevar libro mayor de cuentas, ingresos, pagos, etc

RRHH - Gestión del capital humano, gestión de CV, gestión de vacaciones

Verticales - Software de un rango concreto

SIS - Student Information System - Sistema de gestión de la información del alumno para centros de formación para realizar seguimiento del alumnado, faltas, notas, matriculas, observaciones, 

LMS - Learning Management System - Sistemas de gestión del aprendizaje (intranets de alumnos)

Muy verticales

BIM - Builing Information Model - Sistema de información de edificación para arquitectos y profesionales relacionados
