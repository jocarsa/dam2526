Aplicación triple

Será una aplicación para realización de exámenes y encuestas
1.-Una parte vista hacia el usuario final (alumno o encuestado)
2.-Panel de estadísticas
3.-Panel de administración (preguntas y respuestas)

-Usuario alumno, rellena encuestas
-Usuario espectador, muestra estadísticas
-Usuario admin, crear encuestas o examenes
