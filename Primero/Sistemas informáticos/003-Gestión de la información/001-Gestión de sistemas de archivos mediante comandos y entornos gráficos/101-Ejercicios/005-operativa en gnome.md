Boton derecho sobre escritorio - crear carpeta nueva
Atajo de teclado:
Control + Mayusculas + N = Crear nueva carpeta

En la mayoría de sistemas de ventanas no se puede crear un archivo nuevo (en Windows si que se puede)
Pero podemos crearlo por ejemlpo con el editor de notas, y trabajar con ello:

Copiar cortar y pegar:
Puede hacerse con el boton derecho del ratón:
O bien podemos hacerlo con los atajos de teclado:
Control + C = Copiar
Control + X = Cortar
Control + V = Pegar

Eliminar:
Opcion boton derecho - mover a la papalera
O bien pulsar el boton Supr del teclado 

Eliminar carpeta se hace igual que eliminar archivo:

Opcion boton derecho - mover a la papalera
O bien pulsar el boton Supr del teclado
