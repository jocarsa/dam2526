ls = listado de elementos en el directorio actual

ls -l = listado con detalles

En windows, el listado es con dir

Crear una carpeta nueva 

mkdir micarpeta / mkdir = make directory
cd micarpeta  / cd = change directory

Crear un archivo:
Solo crearlo: touch miarchivo.txt
Crearlo y nos metemos a editarlo: nano miarchivo.txt

En nano:
Control + O = Guardar
Control + X = Salir
