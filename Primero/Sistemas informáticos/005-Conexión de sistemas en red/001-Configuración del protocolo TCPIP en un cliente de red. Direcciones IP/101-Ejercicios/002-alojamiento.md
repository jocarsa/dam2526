2.-Un alojamiento

Un alojamiento es un servidor conectado a internet las 24 horas del dia, ofreciendo un servicio web

3 tipos:
-Alojamiento compartido, bajo rendimiento, bajo control, pero bajo coste - apropiado para proyectos de inicio
Las tecnologías son fijas: HTML, CSS, JS, PHP, MySQL

-Alojamiento VPS (Virtual Private Server) para proyectos intermedios (rendimiento intermedio, mayor control, ultimamente, no mucho más coste, aunque hay que configurarlo)
Las tecnologías son variables: HTML, CSS, JS, PHP, MySQL, Python, node, Mongodb... lo que tu quieras

-Alojamiento dedicado: una máquina física, real, conectada a internet - rendimiento máximo, control máximo, precio elevado
Las tecnologías son variables: HTML, CSS, JS, PHP, MySQL, Python, node, Mongodb... lo que tu quieras
