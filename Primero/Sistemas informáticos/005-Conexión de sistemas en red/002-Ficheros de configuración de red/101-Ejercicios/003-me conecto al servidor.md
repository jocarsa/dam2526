Una vez que tienes un dominio y un servidor, puedes hacer que las DNS del dominio apunten al servidor:

DNS = Domain Name Server - la IP a la que llama el dominio cuando alguien lo llama a él.

IP de mi servidor: 217.160.228.9

El dominio debe apuntar al servidor para que las dos piezas estén atadas

para acceder al servidor:

ssh usuario@maquina
